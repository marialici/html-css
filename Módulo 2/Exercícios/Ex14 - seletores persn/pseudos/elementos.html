Capítulo de livro

Os pseudo-elementos em CSS permitem estilizar partes específicas de um elemento sem a necessidade de adicionar elementos HTML extras. Eles são úteis para aplicar estilos a partes do conteúdo, como a primeira letra ou linha de um parágrafo, ou para inserir conteúdo antes ou depois de um elemento.
